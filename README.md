EdTech Chatbot

Project Overview

This is a simple AI-based chatbot developed using Flask.
It helps users find information about colleges, courses, and education options.

---

Features

- Responds to user queries
- Suggests colleges based on category (engineering, arts, medical)
- Location-based filtering 
- Simple and interactive web interface

---

Technologies Used

- Python
- Flask
- HTML/CSS
- JSON (for dataset)

---

Project Structure

- "app.py" → Main Flask application
- "chatbot.py" → Chatbot logic
- "dataset_generator.py" → Dataset creation script
- "data/" → Stores chatbot data
- "templates/" → Frontend HTML files
- "requirements.txt" → Dependencies

---

 How to Run

1. Clone the repository

git clone <your-repo-link>
cd edtech-chatbot

2. Create virtual environment

python -m venv venv
venv\Scripts\activate

3. Install dependencies

pip install -r requirements.txt

4. Run the app

python app.py
